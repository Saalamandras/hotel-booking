# 🍳 Hotel Breakfast Booking App

A full stack web application for managing hotel breakfast reservations, built as a portfolio project.

## Live Demo
🌐 [hotel-booking-ivory-xi.vercel.app](https://hotel-booking-ivory-xi.vercel.app)

## What it does
- Book breakfast time slots for hotel guests
- Manage capacity limits per slot (max 24 guests)
- Cancel existing bookings
- Bookings persist permanently via a cloud database
- Real-time capacity tracking across all slots

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React |
| Database | Supabase (PostgreSQL) |
| Deployment | Vercel |
| Version Control | Git + GitHub |

## Features
- 4 time slots: 7:00–8:00, 8:00–9:00, 9:00–10:00, 10:00–11:00
- Room number and guest count per booking
- Conflict detection — one booking per room
- Booking log with timestamps
- Responsive design

## Future improvements
- Admin dashboard
- Email confirmations
- Multi-day stay support
- Authentication for staff access
